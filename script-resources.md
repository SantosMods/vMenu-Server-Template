# Script Resources

A curated list of reputable script providers and open-source frameworks for FiveM development.

## Paid Script Providers
- **Night's Software**  
  Premium police, utility, and roleplay scripts  
  <https://store.nights-software.com/>

- **BigDaddy Scripts**  
  High-performance civilian and emergency service scripts  
  <https://bigdaddyscripts.com/>

## Open-Source Frameworks & Utilities
- **Community Ox**  
  A community-driven effort to maintain Overextended resources such as `ox_lib`, `ox_inventory`, and `ox_core` after the original project was archived.  
  <https://coxdocs.dev/>

- **ND Framework**  
  Lightweight alternative to ESX/QBCore with modular resources written in Lua and TypeScript  
  <https://github.com/ND-Framework>
